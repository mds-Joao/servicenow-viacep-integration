# User Story

Como usuário

Quero informar um CEP

Para que os dados do endereço sejam preenchidos automaticamente.

## Critérios de Aceite

* O CEP deve ser validado.
* A consulta deve ser realizada utilizando a API ViaCEP.
* Os campos do endereço devem ser preenchidos automaticamente.
* Deve haver tratamento para CEP inválido.
* Deve haver tratamento para API indisponível.
